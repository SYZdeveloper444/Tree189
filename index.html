Mengapa halaman ini ada?

Halaman ini berfungsi sebagai file index.html yang secara otomatis mengarahkan pengunjung ke home.html. Hal ini diperlukan karena server web biasanya menampilkan index.html sebagai halaman pertama saat sebuah website diakses.

Jika pengalihan tidak bekerja secara otomatis, silakan klik tombol di bawah ini:

Pergi ke Home.html

Informasi Teknis:

Halaman ini menggunakan tiga metode pengalihan untuk memastikan pengunjung sampai ke home.html:

  1. Meta Refresh - Pengalihan otomatis melalui tag meta
  2. JavaScript Redirect - Pengalihan melalui JavaScript jika meta tag tidak bekerja
  3. Link Manual - Tombol untuk pengalihan manual jika kedua metode di atas gagal
